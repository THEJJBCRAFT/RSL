    Wichtig
    
        
    Das Tool erzeugt Java-Commands fuer haeufige Faelle. Sehr spezielle NBT-, Component- und Snapshot-Aenderungen koennen spaeter noch als eigene Presets erweitert werden.
    
      
    
      
    
        
    Keine Assets
    
        
    Es werden keine Minecraft-Bilder, Sounds oder Modelle kopiert. Das Tool nutzt nur Text-IDs fuer die Suche und baut daraus deine Commands.
    
      
    
      
    
        
    Naechster Ausbau
    
        
    Items, Bloecke, Partikel, Entities, Effekte, Enchantments, Attribute und Item Components sind jetzt suchbar bzw. waehbar. Der Generator passt Component- oder NBT-Ausgabe an die Version an.
