RK

RSLRanks v1.0.0 Paper / Spigot

Rang- und Rechte-System mit Vault-, TAB- und PlaceholderAPI-Support, Prefixen, Vererbung und persistenten Spielerdaten.

Ranks Permissions Vault TAB PlaceholderAPI 1.20.1 – 1.21.x
Download · 36 KB

So funktioniert es

Jeder Spieler wird mit UUID, letztem Namen, Rang und Extra-Permissions gespeichert. Beim Join baut RSLRanks ein PermissionAttachment aus Rang, Vererbung und zusaetzlichen Rechten frisch auf.

Installation

Jar in den plugins Ordner legen, Server starten, config.yml anpassen und mit /rank reload neu laden.

Befehle

/rank helpAlle Befehle anzeigen
/rank user <spieler> set <rang>Rang setzen
/rank reloadConfig neu laden
/rank tabconfigTAB-Reihenfolge ausgeben

Permissions

rslranks.admin gibt vollen Verwaltungszugriff. OP-Spieler koennen Raenge vergeben, wenn die Server-Regeln es erlauben.

Arbeitet zusammen mit

Vault (Gruppen & Prefixe), TAB (Tablisten-Sortierung), PlaceholderAPI: %rslranks_rank%, %rslranks_prefix%, %rslranks_priority%, %rslranks_color%.

Fuer maximale Funktion

Vault und TAB installieren, dann /rank tabconfig nutzen und die ausgegebene Gruppenreihenfolge in TAB uebernehmen.

RSLBalance v1.0.0 Paper / Spigot

Eigene Vault-Economy fuer Kontostand, Admin-Geldbefehle und Balance-Anzeige in der Tabliste.

Economy Vault TAB PlaceholderAPI Admin-Tools
Download · 20 KB

So funktioniert es

RSLBalance registriert eine Economy ueber Vault. Andere Plugins koennen dadurch Geldwerte lesen, setzen oder anzeigen, sobald sie Vault-Economy nutzen.

Installation

Zuerst Vault installieren, dann RSLBalance in den plugins Ordner legen und Server starten.

Befehle

/rslbalanceEigenen Kontostand
/rslbalance <spieler>Kontostand anderer
/rsleco set <spieler> <betrag>Betrag setzen
/rsleco add · takeGeld geben / nehmen
/rsleco reloadNeu laden

Permissions

rslbalance.admin fuer Admin-Befehle. OPs erhalten die Rechte automatisch.

Arbeitet zusammen mit

Vault, TAB und PlaceholderAPI. TAB zeigt %vault_eco_balance_formatted%; PlaceholderAPI bietet %rslbalance_balance% und %rslbalance_balance_formatted%.

Fuer maximale Funktion

RSLBalance sollte die aktive Vault-Economy sein. Bei mehreren Economy-Plugins muss klar sein, welches Vault zuerst belegt.

VIDEO
RSLBalance Setup kommt bald

Vault verbinden, Balance in TAB anzeigen und Admin-Geldbefehle nutzen.

RSLWelcome v1.0.0 Paper / Spigot

Animierte Join-Titles mit Schreibeffekt fuer den Spielernamen, Sound, privater Willkommensnachricht und eigener Servername-Datei.

Join-Effekt Title-Animation Sound Standalone
Download · 16 KB

So funktioniert es

Beim Join startet nach kurzer Verzoegerung ein Title: erst WILLKOMMEN, dann wird der Spielername Buchstabe fuer Buchstabe geschrieben. Optional spielt das Plugin Sounds ab.

Installation

Jar in plugins legen, Server starten und RSLWelcome/config.yml sowie server-name.yml anpassen.

Befehle

/rslwelcome reloadConfig neu laden
/rslwelcome testEffekt selbst ansehen

Permissions

rslwelcome.admin erlaubt Reload und Test. OPs haben die Rechte automatisch.

Arbeitet zusammen mit

Laeuft allein. Passt optisch zu TAB, RSLRanks, RSLBalance und dem Redstone-Labs Design.

Fuer maximale Funktion

Servernamen in server-name.yml fest eintragen, damit kein Hoster-Name aus der MOTD erscheint.

VIDEO

SP

RSLSpawn v1.0.0 Paper / Spigot

Fester Join-Spawn mit eigenem /spawn Befehl und optionalem Multiverse-Core Support.

Spawn Join-Teleport Multiverse optional
Download · 8 KB

So funktioniert es

RSLSpawn speichert Welt, Position, Blickrichtung und Pitch. Spieler nutzen /spawn, und beim Join kann der feste Spawn automatisch greifen.

Installation

Jar in plugins legen, Server starten, in die Spawn-Welt gehen und /rslspawn set ausfuehren.

Befehle

/spawnZum Spawn
/rslspawn setSpawn setzen
/rslspawn setworld <welt>Spawn-Welt setzen
/rslspawn tp · status · reloadVerwaltung

Permissions

rslspawn.spawn (Standard fuer alle), rslspawn.admin (Setzen & Reload), rslspawn.tp fuer /rslspawn tp.

Arbeitet zusammen mit

Multiverse-Core optional. Hat MV eine Spawn-Welt geladen, nutzt RSLSpawn einfach den gespeicherten Welt-Namen.

Fuer maximale Funktion

Die Spawn-Welt muss beim Join geladen sein. Bei Multiverse zuerst Welt laden, dann /rslspawn set oder /rslspawn setworld spawn.

RSLNavigator v1.0.0 Flaggschiff Paper / Spigot

Kompletter Server-Navigator: Chat-Trigger, Untermenue-Wizard, voller Item- und Schrift-Designer, Command-Browser und fertige Command-Layouts.

GUI-Editor Item-Designer Command-Browser Untermenues No-Config moeglich
Download · 80 KB

So funktioniert es

Spieler schreiben . im Chat oder nutzen /. — das Navigator-GUI oeffnet sich. Admins bauen Menues, Items, Untermenues und Commands direkt im Editor; Items kommen aus Kategorien, Namen und Lore werden per GUI gestaltet.

Installation

Jar in plugins legen, Server starten und mit /rslnav admin den Editor oeffnen.

Befehle

/rslnav adminAdmin-GUI
/rslnav edit mainHauptmenue bearbeiten
/rslnav open [menu]Menue oeffnen
/rslnav save · reloadSpeichern / neu laden

Permissions

rslnavigator.open fuer Spieler, rslnavigator.admin fuer Editor und alle Verwaltungsbefehle.

Arbeitet zusammen mit

Multiverse-Core optional fuer Welt-Navigation per /mvtp. Verbindbar mit Spawn-, Shop-, Warp- und anderen Plugins per Befehl.

Fuer maximale Funktion

Fast alles laeuft ohne Config-Datei. Der Item-Designer bietet Minecraft-Farben, Styles, Hex-Codes, RSL-Presets und Gold-Gradient. Bekannte Commands (teleport, give, gamemode, effect, mvtp, spawn, warp/home, msg) bekommen passende Layouts.

RSLCrates v1.0.0 17 Animationen Paper / Spigot

Crate-Key-Plugin mit GUI-Baukasten, 12 Varianten, 17 Mega-Animationen, Effekt-Staerken, Overlap-Schutz und Name-Designer.

Crates / Keys GUI-Baukasten Animationen Vault optional Rewards
Download · 84 KB

So funktioniert es

Admins schauen auf einen Block und erstellen daraus mit /rslcrates create <id> eine Crate. Doppelte Crates auf demselben Block werden blockiert; Loeschen und Name-Design laufen ueber GUI.

Installation

Jar in plugins legen. Fuer Geld-Oeffnung zusaetzlich Vault plus RSLBalance (oder andere Vault-Economy) nutzen.

Befehle

/rslcrates adminGUI-Baukasten
/rslcrates create <id>Crate erstellen
/rslcrates edit <id>Crate bearbeiten
/rslcrates key give <spieler> <crate> [n]Keys vergeben
/rslcrates reload · saveVerwaltung

Permissions

rslcrates.open (Spieler), rslcrates.admin (Baukasten), rslcrates.key.give (Key-Ausgabe).

Arbeitet zusammen mit

Vault & RSLBalance fuer Preise, Paid-Crates, Jackpot-Geld, Streak-Bonus und Money-Rewards. Ohne Economy laufen Key-/Free-Crates und Item-/Command-/Message-Rewards weiter.

Varianten & Effekte

12 Varianten: KEY, PAID, KEY_OR_PAID, FREE, TIMED, PREVIEW, MYSTERY, EVENT, VIP, JACKPOT, STREAK, LEVEL. Effekt-Staerke: CALM, NORMAL, EPIC.
