Plugin

RSLRanks

Rang- und Rechte-System fuer Paper/Spigot 1.20.1 bis 1.21.x mit Vault, TAB und PlaceholderAPI Support.

RSLRanks downloaden

So funktioniert es

RSLRanks speichert Spieler mit UUID, letztem Namen, Rang und Extra-Permissions. Beim Join bekommt der Spieler ein PermissionAttachment, das aus Rang, Vererbung und Extra-Rechten neu aufgebaut wird.

Installation

Jar in den Paper/Spigot `plugins` Ordner legen, Server starten, `config.yml` anpassen und danach `/rank reload` ausfuehren.

Befehle

`/rank help`, `/rank user <spieler> set <rang>`, `/rank reload`, `/rank tabconfig`.

Permissions

`rslranks.admin` gibt vollen Verwaltungszugriff. OP-Spieler koennen Raenge vergeben, wenn die Server-Regeln es erlauben.

Arbeitet zusammen mit

Vault fuer Gruppen und Prefixe, TAB fuer Tablisten-Sortierung, PlaceholderAPI fuer `%rslranks_rank%`, `%rslranks_prefix%`, `%rslranks_priority%` und `%rslranks_color%`.

Wichtig fuer maximale Funktion

Vault und TAB installieren, dann `/rank tabconfig` nutzen und die ausgegebene Gruppenreihenfolge in TAB uebernehmen.

Plugin

RSLBalance

Eigenes Vault-Economy-Plugin fuer Kontostand, Admin-Economy und TAB Balance-Anzeige.

RSLBalance downloaden

So funktioniert es

RSLBalance registriert eine Economy ueber Vault. Andere Plugins koennen dadurch Geldwerte lesen, setzen oder anzeigen, wenn sie Vault-Economy nutzen.

Installation

Vault zuerst installieren, dann RSLBalance in den `plugins` Ordner legen und Server starten.

Befehle

`/rslbalance`, `/rslbalance <spieler>`, `/rsleco set <spieler> <betrag>`, `/rsleco add`, `/rsleco take`, `/rsleco reload`.

Permissions

`rslbalance.admin` fuer Admin-Befehle. OPs bekommen die Admin-Rechte automatisch.

Arbeitet zusammen mit

Vault, TAB und PlaceholderAPI. TAB kann `%vault_eco_balance_formatted%` anzeigen; PlaceholderAPI kann `%rslbalance_balance%` und `%rslbalance_balance_formatted%` nutzen.

Wichtig fuer maximale Funktion

RSLBalance sollte die aktive Vault-Economy sein. Wenn ein anderes Economy-Plugin installiert ist, muss klar sein, welches Plugin Vault zuerst nutzt.

Plugin

RSLWelcome

Animierte Join-Titles mit Sound, privater Willkommensnachricht und eigener Servername-Datei.

RSLWelcome downloaden

So funktioniert es

Beim Join startet nach kurzer Verzoegerung ein Title: erst WILLKOMMEN, danach wird der Spielername Buchstabe fuer Buchstabe geschrieben. Optional spielt das Plugin Sounds ab.

Installation

Jar in den `plugins` Ordner legen, Server starten und `plugins/RSLWelcome/config.yml` sowie `server-name.yml` anpassen.

Befehle

`/rslwelcome reload` laedt die Config neu. `/rslwelcome test` zeigt den Effekt fuer dich selbst.

Permissions

`rslwelcome.admin` erlaubt Reload und Test-Befehl. OPs haben diese Rechte automatisch.

Arbeitet zusammen mit

Laeuft alleine. Passt optisch zu TAB, RSLRanks, RSLBalance und dem NeyexSMP/Redstone-Labs Design.

Wichtig fuer maximale Funktion

Den Servernamen in `plugins/RSLWelcome/server-name.yml` fest eintragen, damit kein Hoster-Name aus der MOTD angezeigt wird.
